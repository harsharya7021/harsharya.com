How do I make a key?
  1. GitHub → Settings → Developer settings → Personal access tokens → Fine-grained → Generate new.
  2. Resource owner harsharya7021 · Repository access → Only select → harsharya.com.
  3. Permissions → Contents → Read and write. Generate, copy, paste above.
  4. Keep a copy in your password manager. The key stays only on this device.
Not set up yet?
  1. The server is optional — the Publish key tab works today with zero setup.
  2. To route through your VPS instead, deploy _os/publish-service (see its README), point os.harsharya.com at the box, then sign in here.
  3. With the server, your GitHub token lives on the box — never in this browser.
